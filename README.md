# Website Panduan Fashion "Fashion.Instant"

## Deskripsi
Website Panduan Fashion "Fashion.Instant" merupakan website yang dirancang untuk memberikan informasi dan inspirasi seputar dunia fashion. Website ini membantu pengunjung menemukan referensi gaya berpakaian melalui artikel, tips fashion, serta informasi yang dikemas dalam tampilan sederhana, modern, dan mudah diakses. Tujuannya adalah agar pengguna dapat meningkatkan rasa percaya diri dalam berpenampilan tanpa harus mengeluarkan biaya yang besar.

## Bahasa Pemrograman
- HTML
- CSS

## Fitur
- Hero Section sebagai halaman utama dengan tampilan menarik.
- Navigation Bar yang memudahkan pengguna berpindah ke setiap halaman.
- Halaman Tentang yang menjelaskan tujuan dan informasi mengenai Fashion.Instant.
- Halaman Artikel yang berisi berbagai informasi dan panduan seputar fashion.
- Halaman Kontak yang menyediakan informasi untuk menghubungi pengelola website.
- Responsive Design sehingga website tetap nyaman digunakan pada berbagai ukuran layar.
- User Interface (UI) yang sederhana, modern, dan mudah dipahami.

## Cara Menjalankan
1. Download atau clone repository ini.
2. Buka folder `all_files`.
3. Jalankan file `index.html` menggunakan browser.
4. Website siap digunakan.

## Tujuan Project
Project ini dibuat sebagai media informasi dan edukasi mengenai dunia fashion agar masyarakat dapat memperoleh inspirasi gaya berpakaian, mengikuti tren fashion, serta meningkatkan kepercayaan diri dalam berpenampilan.
